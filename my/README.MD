# Приложение my
Приложение охватывает функционал личного кабинета пользователя/администратора.

Структура приложения:
1. Модели:

   1.1. Корзина

   1.2. Заказ

   1.3. Позиция заказа

   1.4. Профиль пользователя

   1.5. Сообщение

2. Представления (папка views)
3. Сервисы (папка services)
4. URLы (urls.py)
5. Таблицы (файл tables.py)
